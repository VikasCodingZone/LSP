
# 🎓 Campus Wallet

Campus Wallet is a secure and modern **campus-based payment and transaction system** designed for students and institutions to manage digital payments efficiently.

---

## 🚀 Features

- 🔐 Secure Authentication (JWT based login system)
- 📧 OTP-based Password Reset via Email
- 💳 Digital Wallet System for campus payments
- 🏫 Student-friendly UI for easy navigation
- 🔄 Forgot & Reset Password flow
- 📊 Clean and scalable backend architecture (Node.js + Express)
- 🗄️ MongoDB database integration

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- JavaScript (JSX)
- CSS

**Backend:**
- Node.js
- Express.js
- MongoDB
- Mongoose

**Other Tools:**
- Nodemailer (Email service)
- JWT (Authentication)
- dotenv (Environment config)
